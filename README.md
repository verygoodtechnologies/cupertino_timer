# cupertino_timer

An iOS style timer widget.

## Usage


### Basic
```
CupertinoTimer(duration: new Duration(minutes 1))
```

### Advanced
```
CupertinoTimer(
    duration: new Duration(minutes 1),
    timeStyle: new TextStyle(fontSize: 24),
    startOnInit: true,
    ringColor: Colors.blue,
    ringStroke: 2)
```
